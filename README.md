Delivery Reliability Dashboard

Power BI dashboard focused on delivery performance and logistics reliability analysis.

Overview

This project analyzes delivery operations using a synthetic logistics dataset.
The dashboard provides insights into:

On-time delivery rate

Late deliveries analysis

Courier performance

Delivery types comparison

City and depot distribution

Responsibility breakdown for delays

Tools

Power BI

DAX

Data Modeling

Dataset

The dataset used in this project is synthetic and was provided as part of a test assignment.
No real customer or confidential company data is included.

Purpose

Portfolio project demonstrating skills in:

Business Intelligence

KPI design

Data visualization

Logistics analytics

Performance monitoring dashboards
